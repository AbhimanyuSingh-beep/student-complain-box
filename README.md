# student-complain-box
Student Complaint Box
A lightweight Python command-line application for managing student complaints in educational institutions. This system enables students to submit complaints anonymously and allows administrators to track, categorize, and manage them efficiently.
Overview
The Student Complaint Box is a simple yet effective tool designed to provide a formal channel for students to voice their concerns. Students can submit complaints across multiple categories, and administrators have full control to view, organize, and update complaint statuses.
Features

✅ Easy Complaint Submission - Students can submit complaints in just a few steps
✅ Six Complaint Categories - Academic, Facilities, Faculty, Staff, Admin, Other
✅ Unique Complaint ID - Each complaint gets an auto-generated ID for tracking
✅ Status Tracking - Monitor complaint progress with customizable statuses
✅ View All Complaints - Administrators can see complete complaint details
✅ Category-Based Organization - Filter complaints by category with statistics
✅ Update Complaint Status - Change status as complaints are addressed
✅ Delete Complaints - Remove complaints from the system
✅ Role-Based Interface - Separate student and admin menus
✅ No Dependencies - Uses only Python standard library
